A utility R package providing ~50 functions for data analysis and visualization. Covers package management, color palette tools, statistical testing (t-test, ANOVA, chi-square, correlation, power analysis), data visualization (bar, pie, density, Venn, forest plots), bioinformatics utilities (gene ID conversion, GMT parsing, GEO access), and custom infix operators. Follows tidyverse principles.
